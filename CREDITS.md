## phpqrcode

**phpqrcode** is used for generating QR codes in PNG format.

- Project: phpqrcode
- Author: Dominik Dzienia
- License: GNU LGPL
- Website: http://phpqrcode.sourceforge.net/

phpqrcode is a lightweight, dependency-free PHP library that enables reliable
QR code generation and is well-suited for long-term, server-side applications.

## Gutendex

**Gutendex** is a simple, self-hosted web API for serving book catalog information
from **Project Gutenberg**, an online library of free eBooks.

- Project: Gutendex
- Author: Gareth Johnson
- License: MIT License
- Copyright © 2017 Gareth Johnson
- Repository: https://github.com/garethbjohnson/gutendex

Gutendex is used as a data source for publicly available bibliographic metadata.
